# Installation guide

```sh
# reference site : https://trufflesuite.com/boxes/react/
$ npm install -g truffle

$ mkdir NFT_truffle

$ cd NFT_truffle

$ truffle unbox react
```

```sh
# Alternatively, run `truffle unbox` via npx
$ npx truffle unbox react
```

# reference site : https://docs.openzeppelin.com/contracts/4.x/erc721
